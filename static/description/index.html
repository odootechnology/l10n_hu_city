Hungarian Address extension Database

This module provides Odoo with a complete list of Hungarian cities.

There is no need to type in any city manually anymore.

The database contains all cities, including their zip codes, and can be used in the contact form.

What can you do with this module?

  • start typing the city name for easy selection
  • type in the zip codes, and gives back the city and County too
  • the module automatically fills in the county name based on the city name

Origin/Forrás: Hungarian Post website up to date: 2025.05.27
